Instruções de uso:
1) Extrair o zip "calcados-miranda", entrar no diretório "public_html" e colar a pasta extraida.
2) Entrar em "http://localhost/phppgadmin/", criar um banco de dados com o nome "calcadosmiranda" e importar o arquivo .sql contido na pasta "Script SQL" para rodar o script de comandos de criação do banco de dados e inserção de dados.
3) Rodar a aplicação através de "http://localhost/~nomedeusuario/calcados-miranda/"


OBS: A página do admin se encontra no caminho ".../calcados-miranda/admin/login.php"
